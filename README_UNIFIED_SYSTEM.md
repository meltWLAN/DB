# 股票分析系统 - 统一入口

这个统一启动器集成了系统中所有功能模块，提供了一个单一的桌面入口，让您可以便捷地访问所有分析功能。

## 功能特点

- **单一入口**：通过一个界面即可访问所有分析模块
- **模块化设计**：保持各个功能模块的独立性
- **状态检查**：自动检查各模块的可用性
- **简洁界面**：直观的卡片式设计，易于导航

## 包含的分析模块

1. **主分析系统**
   - 基础动量分析
   - 均线交叉策略
   - 财务分析
   - 筹码分析

2. **机器学习分析系统**
   - 基于机器学习的动量模型
   - 特征重要性分析
   - 市场状态自适应

3. **热门股票分析系统**
   - 连续涨停股扫描
   - 潜在暴涨股分析
   - 热门板块识别
   - 涨停延续性预测

4. **增强API系统**
   - 增强的数据获取和处理
   - 更可靠的API连接
   - 高质量数据筛选

## 如何使用

1. **启动系统**：
   ```
   python start_unified_system.py
   ```
   或直接双击 `start_unified_system.py` 文件

2. **选择模块**：
   在启动界面中，您将看到所有可用的分析模块，点击相应的"启动"按钮即可打开所需模块

3. **使用模块**：
   每个模块启动后，将在单独的窗口中运行，您可以同时运行多个模块

## 系统要求

- Python 3.6+
- 所需Python库:
  - tkinter
  - pandas
  - numpy
  - matplotlib
  - scikit-learn (机器学习模块需要)
  - PIL/Pillow (图标显示需要)

## 注意事项

- 统一入口会检查各个模块的依赖文件，如有缺失将会禁用相应模块
- 如需添加新模块，请在 `unified_launcher.py` 中的 `MODULES` 列表中添加配置
- 主窗口会在启动模块后最小化，但不会关闭，您可以随时恢复窗口启动其他模块

## 故障排除

如果遇到启动问题:

1. 检查日志文件 (logs目录下)
2. 确保所有依赖库已安装
3. 验证所有模块文件都存在于正确位置

## 自定义

您可以通过编辑 `unified_launcher.py` 文件来调整界面外观和行为:

- 修改窗口大小
- 调整颜色主题
- 添加或移除模块
- 自定义模块描述和图标
